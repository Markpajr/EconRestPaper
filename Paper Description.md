# EconRestPaper
An Analysis of the Relationship between Producer Price Indexes and Restaurant Industry Stock Values, with statistical testing conducted with R.
Testing done included model misspecification, heteroscedasticity, multicollinearity, omitted variable bias, and autocorrelation.
This was my first exploration into writing code for R, and sparked my interest in its possibilites.
